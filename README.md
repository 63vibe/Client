
  # AI Newsletter for Company Bulletin

  This is a code bundle for AI Newsletter for Company Bulletin. The original project is available at https://www.figma.com/design/FfLFucsdyNHafO0RDoyg08/AI-Newsletter-for-Company-Bulletin.

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.

## 이모부에게 사이트 보여주기 (가장 빠른 방법)

현재 프로젝트는 `/inventory-mvp` 데모 화면만 먼저 보여주면 되므로, **Vercel 무료 플랜**으로 바로 공개 링크를 만들 수 있습니다.

1. GitHub에 현재 브랜치 푸시
2. Vercel에서 저장소 Import
3. Build Command: `npm run build`, Output: Next.js 기본값 사용
4. 배포 후 생성된 URL에서 `https://.../inventory-mvp` 접속

데모 로그인 정보:
- 아이디: `abcd`
- 비밀번호: `abcd!!`

> 참고: Supabase 환경 변수가 없어도 `/inventory-mvp` 데모는 확인 가능하도록 처리되어 있습니다.
